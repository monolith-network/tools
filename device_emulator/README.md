# Device Emulator (demu)


The demu application can take the following sensors :

```
   "temperature", 
   "motion", 
   "humidity", 
   "air_pressure", 
   "flame",
   "light"
```

into its configuration to poll the given environment which can be generated with the following attributes:

```
   "dry",
   "wet",
   "high_traffic",
   "medium_traffic",
   "low_traffic",
   "hot",
   "cold",
   "temperate",
```
